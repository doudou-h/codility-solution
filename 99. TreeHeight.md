## **[99. TreeHeight](https://app.codility.com/programmers/lessons/99-future_training/tree_height/)**

### Question
Given a non-empty binary tree T consisting of N nodes, returns its height. An empty tree has height âˆ’1.

Constraints:
- `1 <= N <= 1,000`
- `0 <= height <= 500`

### Solution

#### Python
```python
from extratypes import Tree  # library with types used in the task

def solution(T):
    def height(T: Tree):
        if not T: return -1
        return 1 + max(height(T.l), height(T.r))
    return height(T)
```
